The FilterSelect component is used to create a select dropdown for a filter based on passed options.

```js
    const options = [
      {
        label: 'Option A',
        value: 'Option A'
      },{
        label: 'Option B',
        value: 'Option B'
      },{
        label: 'Option C',
        value: 'Option C'
      }
    ];
	<FilterSelect options={options} icon="tag" />
```
