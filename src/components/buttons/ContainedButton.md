The ContainedButton component is a larger button to be used for forms.

```js
  <h4>Base Palette Super Buttons</h4>
  <ContainedButton role="button" aria-label="contained default button" btnStyle="default" onClick={()=>{}} />
  <ContainedButton role="button" aria-label="contained primary button" btnStyle="primary" onClick={()=>{}} />
  <ContainedButton role="button" aria-label="contained secondary button" btnStyle="secondary" onClick={()=>{}} />
  <ContainedButton role="button" aria-label="contained tertiary button" btnStyle="tertiary" onClick={()=>{}} />
```
